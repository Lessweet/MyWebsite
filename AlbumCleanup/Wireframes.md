## 配色方案 (Color Scheme)

> 极简 6 色系统，通过透明度变化实现所有视觉层次

### 语义化颜色命名 (Color+Theme.swift)

```swift
// ═══════════════════════════════════════════════════════════════
//  PureShot 极简色彩系统 - 6 色 + 透明度变体
// ═══════════════════════════════════════════════════════════════

extension Color {

    // ─────────────────────────────────────────────────────────────
    //  1. 背景色 (Background)
    // ─────────────────────────────────────────────────────────────
    static let psBackground = Color("Background")
    //  Light: #FAFAFA (近白)
    //  Dark:  #0A0A0A (近黑)

    // ─────────────────────────────────────────────────────────────
    //  2. 表面色 (Surface) - 卡片、浮层
    // ─────────────────────────────────────────────────────────────
    static let psSurface = Color("Surface")
    //  Light: #FFFFFF
    //  Dark:  #1C1C1E

    // ─────────────────────────────────────────────────────────────
    //  3. 主文字 (Text Primary)
    // ─────────────────────────────────────────────────────────────
    static let psTextPrimary = Color("TextPrimary")
    //  Light: #1A1A1A
    //  Dark:  #F5F5F5

    // ─────────────────────────────────────────────────────────────
    //  4. 次级文字 (Text Secondary)
    // ─────────────────────────────────────────────────────────────
    static let psTextSecondary = Color("TextSecondary")
    //  Light: #6B6B6B
    //  Dark:  #8E8E93

    // ─────────────────────────────────────────────────────────────
    //  5. 强调色 (Accent) - Cream 奶油色 ⭐ 品牌色
    // ─────────────────────────────────────────────────────────────
    static let psAccent = Color("Accent")
    //  统一: #F5E6D3 (Cream)
    //  用途: 光波、高亮、选中态、品牌标识

    // ─────────────────────────────────────────────────────────────
    //  6. 警示色 (Destructive) - 删除操作
    // ─────────────────────────────────────────────────────────────
    static let psDestructive = Color("Destructive")
    //  统一: #FF6B6B (柔和红)
    //  用途: 删除数量、警示信息
}
```

### 透明度变体规则

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  【透明度语义】                                                  │
│                                                                 │
│  100%  ━━━━━━━━━━━  实体/焦点元素                               │
│   80%  ━━━━━━━━     强调/高亮                                   │
│   40%  ━━━━         中等/过渡态                                 │

│                                                                 │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  【应用示例】                                                    │
│                                                                 │
│  psAccent.opacity(1.0)   → 选中照片边框                         │
│  psAccent.opacity(0.8)   → 光波核心                             │
│  psAccent.opacity(0.4)   → 光波中层                             │
│  psAccent.opacity(0.15)  → 光波边缘/背景光晕                    │

```

## 液态透镜材质 (Liquid Lens Material)

> 从"玻璃卡片"进化到"液态透镜"，引入表面张力的物理质感

| **长按取消** | 0.3s+0.2s | 长按确认 | **光波收缩 + ✓消失** |
| 滑动切换 | 即时 | 轻触 | 非线性阻尼 |
| 照片融化 | 0.6s | 连续轻震 | 边缘液态化 |
| 液滴流淌 | 0.5s | 轻柔流动 | 向下滴落 |
| 液滴消散 | 0.3s | 轻触 | 淡出消失 |
| 清理完成 | 0.2s | 确认震动 | 保留照片稳定 |
